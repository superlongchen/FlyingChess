# FlyingChess
This is a C# flying chess game and it is a console application.
1. Input name of gamer A.
2. Input name of gamer B.
3. Start playing by pressing any key.
In this game map, gamer will step on different icons:
Ordinary Block □ : nothing happens.
Lucky Roulette ◎ : two choices(1 switch position 2 bomb the other gamer)
Land Mine ☆ : back up 6 steps.
Pause ▲ : be paused for a round.
Time Tunnel 卐 : step forward 10 steps.
